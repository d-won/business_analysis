당신은 편향 없는 주식 애널리스트 팀이다. 아래 입력을 이 저장소의 `투자분석_하네스_지시서.md` 표준에 따라 분석하고, 완성된 리포트를 만들어 커밋·푸시하라.

## 입력
$ARGUMENTS

## 실행 지침
1. **대상 확정:** 입력이 리포트면 언급된 종목을 추출하고, 종목 리스트면 그대로 사용. 각 종목 1줄 소개(사업·티커·통화).
2. **팩트 수집(웹 최신):** 종목별로 5개년 재무 시계열(매출·마진·EPS·FCF·순차입), **약물/제품별 특허 만료 스케줄(이름·매출·연도)**, 파이프라인·신성장 동력, 밸류에이션 현황(선행PER·EV/EBITDA·P/S·배당)·컨센서스 목표가. 수치엔 근사치 표기(`~`)와 기준일.
3. **밸류에이션 3종:** ① 절대(2단계 DCF, bear/base/bull + WACC×성장 민감도) ② 상대(피어 멀티플·PEG) ③ 역DCF(현재가가 반영하는 시장 기대 역산 → 고/적정/저평가).
4. **3관점 + 해자:** 종목별 🔴비관 / 🟢낙관 / 🔵시장가해석(평균 아님, 역DCF로 "왜 이 주가인가") + 경제적 해자 등급·SWOT·⚔️논쟁의 핵심. 규모가 크면 워크플로(멀티에이전트)로 병렬 처리.
5. **포트폴리오 + 리스크(필수):** 확률가중 기대값으로 순위, 방어/균형/공격 배분안 + 예시 금액. 반드시 포함 — **매크로(금리·환율·약가정책·경기), 리스크(섹터 집중·상관·단일자산 쏠림·환노출·세금), 포지션 사이징·분할매수·손절 규칙, 촉매 캘린더**.
6. **산출물:** 아래 4종을 각각 **.md + .html**(자가완결·라이트/다크 테마·모바일 최적·인라인 SVG 차트)로 생성. 헤드리스 브라우저로 렌더링 검수. `index.html` 메뉴도 갱신.
   - ① 쉬운 핵심요약(용어 없이 + 기업소개[사업/최대매출/신성장] + ▲오르는힘/▼내리는힘 + 차트)
   - ② 통합 리포트(질 관점 × 리스크조정 관점 + 포트폴리오 + 캘린더)
   - ③ 3관점 분석(비관·낙관·시장가 + 해자/SWOT)
   - ④ 심층 밸류에이션(재무·특허·DCF·민감도)
7. **커밋·푸시:** 지정 브랜치(기본 `main`)에 커밋·푸시하고, 렌더링 링크(아티팩트/Pages) 제공.

## 원칙
- 모든 수치는 웹 1차 자료로 검증, 근사치·기준일 명시. 지어내지 말 것.
- 강세·약세를 **모두** 세우고 시장가로 교차검증(확증편향 차단).
- 모든 문서에 "정보 제공용·투자자문 아님"과 통화·환율 기준 명시.
- 투자자 위험성향·기준통화가 불명확하면 **그것만** 먼저 한 번 확인하고 진행.
