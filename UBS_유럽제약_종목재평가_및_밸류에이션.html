절대평가 핵심 근거

  1. 특허절벽의 '시점'이 절대가치를 좌우. DCF는 먼 미래 현금흐름을 크게 할인 → 동일 규모라도 AZN(2031~34)은 노바티스(2029~31)보다 훼손이 작다. (AZN>노바티스의 정량 근거)
  2. 특허 무관 사업 = 안정적 터미널가치. 머크·갈더마는 예측기간 이후 가치(DCF의 60~70%) 신뢰도↑. 단 갈더마는 성장률 민감도 큼.
  3. 파이프라인 = rNPV로 가감. 로슈 giredestrant: persevERA 실패로 전이성 1차 감액, 그러나 lidERA(HR 0.70)·evERA(HR 0.38) 성공 + FDA 우선심사로 최대 잠재영역 방어.
  4. 영업레버리지 = 마진 가정의 핵심. 출시 중후반 약물은 추가매출 대부분이 이익 전환(AZN 코어OPM 28→35%). 절벽 도래 시 역회전.
  5. 디레버리징 이익 인플렉션(갈더마). 순차입 2.3→1.5x로 이자부담 감소 → 컨센 EPS +24%/yr. 절대가치는 정상화 속도에 민감.
  6. AI 운영효율 = 옵션가치 소액 가산. 서류·환자모집 각 6개월 단축 → 특허 판매기간 ~1년 연장(+10%). '신약 발견' AI는 아직 base-case 제외.

5-4. 실제 2단계 DCF 계산 결과 (예시적 산출)

위 프레임을 실제 숫자로 계산했습니다. 방법: 기준FCF에 10년 성장 적용(1~5년 g1, 6~10년 g2) → 영구가치 → WACC 할인한 기업가치(EV) − 순차입 = 내재 자기자본가치를 현재 시가총액과 비교(주당 대신 시총 대비 상승여력으로 통일). 성장률은 성장궤적·절벽 타이밍 반영해 차등.

DCF가 말해주는 5가지:
  1. 효율적 시장 — 대형주는 대부분 기준 시나리오에서 현 시총 ±15% 내. 초과수익은 시나리오 실현 확률 게임.
  2. AZN 비대칭성 최우수. 기준 −2%(적정)인데 강세 +46%가 파이프라인·$80b(2030) 목표로 근거 탄탄 → 하방보다 상방 실현확률이 높은 유일 케이스. 절대평가로도 최선호 재확인.
  3. 갈더마 = "좋은 사업, 안전마진 제로" 정량 확인. 강세조차 +6%(시장이 고성장 완전 반영) + 터미널 비중 65~72% 최고 = 먼 미래 의존도 최대 = 가장 위험 → '조정 시 진입' 뒷받침.
  4. 사노피 밸류트랩 정량 확인. Dupixent 절벽을 후반부 성장 0%로 반영하니 기준 −7% → 선행 PER 8배는 저평가가 아니라 절벽 보상.
  5. 로슈 = 밸류+옵션. 기준 −9%로 적정 부근이나 giredestrant 강세(peak CHF5.8b) 실현 시 상방.
