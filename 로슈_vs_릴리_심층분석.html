🇨🇭 로슈 (Roche)

⚡ 결론 — 빅3 바이오시밀러 침식이 종료되며 성장이 순증 국면으로 전환된 "무너지지 않는 배당복리 디펜시브"로, 현재가 CHF338.80은 파이프라인 옵션(비만·치매·유방암)을 거의 공짜로 얹은 적정~소폭 저평가 상태 — 하방은 배당·마진이, 상방은 파이프라인이 담당하는 비대칭 프로파일이다.

핵심 레버 3가지

  • 레버① giredestrant 상업화 규모 — 경구 SERD 유방암, 피크 컨센 강세 ~CHF5.8b vs 약세 ~CHF1.2b(스프레드 4.6b). evERA FDA 결정 2026.12 단일 이벤트로 주가 두 자릿수 변동, 목표가 밴드(230~462)의 넓이를 설명.
  • 레버② LOE 방어(자기잠식 제형화) — Ocrevus(CHF4.9b, 2028~29)·Hemlibra(CHF4.8b)·HER2(~CHF9b) 절벽을 Phesgo(SC 5분)·Ocrevus SC(Zunovo)로 붕괴가 아닌 완충으로 관리. 성패가 4~5% CAGR 성립의 필요조건.
  • 레버③ 저WACC 리레이팅 — 저베타(~0.6~0.7)·39년 연속 증배가 6.5~7.5% WACC를 정당화. TV 비중 70% 구조에서 WACC ±0.5%p가 밸류를 ±13% 흔드는 최대 단일 동인.
1. 사업 개요·수익 구조
  • 두 개의 축, 사실상 제약 회사 — FY25 그룹 매출 ~CHF61.5b는 제약 ~CHF47.7b(~78%)·진단 ~CHF13.8b(~22%). 그룹 코어영업이익 ~CHF22b의 80%대 후반~90%가 제약에서 발생.
  • 이익 편중 — 제약 코어영업이익률(COP) ~49.2% vs 진단 10%대 후반~20% 초반. "진단으로 해자를 넓히고 제약 고마진 바이오로 돈을 번다"는 구조.
  • 제약 = 소수 블록버스터 집중 모델 — 상위 5개(Phesgo·Xolair·Ocrevus·Hemlibra·Vabysmo) 합계 ~CHF21.4b로 제약 매출의 ~45%. 전년 대비 CER 기준 +CHF3.2b 순증.
  • 수익 메커니즘(제약) — (i) 특허·독점기간 동안 고가·고마진, (ii) 제조 난도 높은 바이오의약품이라 시밀러 침투가 저분자보다 느림, (iii) 상당수가 자사 동반진단과 묶여 처방 유도.
  • 진단 = razor-razorblade 모델 — 분석기(instrument)를 깔고 소모품·시약(시장의 ~66%)을 반복 판매하는 고정형 반복매출. 병리·동반진단이 제약 파이프라인과 결합해 고성장.
  • 진단 억제 요인 — 중국 약가/조달개혁(VBP)과 COVID 특수 소멸이 성장률을 CER +1~2%로 눌렀음.

경쟁 구도

  • 안과(Vabysmo) — 리제네론 Eylea/Eylea HD가 최대 적수. wet AMD 점유 ~30%, DME ~22% 상승세, 신규환자 ~50% 흡수.
    • 반격 요인 — Eylea HD가 투여간격 연장으로 반격. 2030년까지 두 진영이 항VEGF-A ~60% 과점 전망.
    • 로슈 우위 — 신규환자·복합기전(VEGF+Ang-2)에서 우위이나 시장 위축으로 성장 둔화.
  • 다발성경화증(Ocrevus) — 노바티스 Kesimpta(월1회 자가 SC)·TG Therapeutics Briumvi(짧은 IV)가 편의성으로 잠식.
    • 방어 — 6개월 1회·PPMS 유일 적응증 + SC 제형 출시로 응수. 편의성 축은 밀리나 임상·데이터·PPMS 독점으로 버팀.
  • HER2 유방암 — 시밀러가 원 IV 침식. Phesgo(SC 5분)로 프랜차이즈 자가 이식하며 방어.
    • 위협 — 아스트라제네카/다이이찌 Enhertu가 차세대 ADC로 판도 흔듦.
  • 혈우병(Hemlibra) — 여전히 표준. 화이자·노보노디스크 차세대 및 유전자치료가 장기 위협.
  • 진단 — 글로벌 IVD 점유 ~20%로 명확한 1위. Abbott·Siemens Healthineers·Danaher(Beckman)·Sysmex·bioMérieux·BD가 후발.
    • 강점 — 코어랩 설치기반·메뉴폭·병리/동반진단(+25%). 약점은 중국 VBP·COVID 특수 소멸.

파이프라인 승부수

  • giredestrant(경구 SERD) — 초기암 lidERA 성공(재발·사망 -30%, FDA ~2026.11 우선심사), 전이성 evERA 성공(everolimus 병용 진행·사망 -44%, ESR1변이 -62%, FDA ~2026.12). 전이성 단독 persevERA는 실패.
    • 의미 — 경구 SERD 최초 curative setting 성공 타이틀로 AZ camizestrant·릴리 imlunestrant 선점 시도. 피크 컨센 ~CHF5.8b vs ~CHF1.2b, 밸류에이션 최대 변수.
  • 비만 CT-388(Carmot, GLP-1/GIP) — 릴리·노보 아성에 늦은 진입. petrelintide(amylin) 병용 등 조합 전략으로 top-3 목표.
  • 치매 trontinemab(Brainshuttle) — 트랜스페린 수용체 이용, 저용량으로 빠르고 강력한 아밀로이드 제거. TRONTIER 1·2 3상(1,600명, 18개국) 개시. Leqembi·Kisunla 대비 뇌전달 효율·ARIA 안전성 차별화.

경제적 해자

  • 종합 판정 — 광폭(wide)에 가까우나 '특허 절벽'이라는 시한부 성분이 항상 섞임. 지속성은 파이프라인 성공률에 종속.
  • ① 무형자산(특허+R&D 생산성, 강·순환적) — 바이오의약품 특허가 고마진 보장. LOE 지형: Ocrevus(~2028–29)·Hemlibra(~2020년대 후반)·Vabysmo(~2032+).
    • 본질 — 특허 자체가 아니라 특허를 계속 갈아끼우는 R&D 성공률이 진짜 해자.
  • ② 규모의 경제+무형자산(진단, 강·지속적) — IVD ~20% 세계 최대 설치기반이 시약 반복매출·유통·규제데이터에서 복제 불가한 비용우위. 제약 특허보다 길고 안정적.
  • ③ 전환비용+네트워크(제약×진단 결합, 강화 방향) — 동반진단으로 자사 치료제 처방 유도하는 수직결합. 병리·동반진단 +25% 성장이 증거.
  • ④ 브랜드/규모(자본배분) — 39년 연속 증배·순부채 축소·CER 두 자릿수 이익성장이 낮은 자본비용 부여, M&A·R&D 재투자로 해자를 자금으로 되사는 선순환.
  • 침식 요인 — 편의성 우위 경쟁자(Kesimpta·Briumvi·Eylea HD), 차세대 ADC(Enhertu), 다가오는 Ocrevus·Hemlibra LOE, CHF 강세·중국 약가.
  • 강화 요인 — 빅3 침식 종료, 초고마진 신약 믹스, 제약×진단 결합 심화, 2026년 다수 FDA 이벤트.
3. 매크로 (요인별)
  • CHF 환율 → 매출 환산(미국 ~50% 노출) + 마진(트랜잭션, 원가 CHF 고정) → 최대 단기 리스크. CHF 10% 강세 시 매출 -3~5%p, 마진 -1~2%p 압축. CER +7%가 보고 +2~3%로 축소, PER 17.5x→15~16x 디레이팅 위험.
  • 금리/WACC → 밸류에이션(파이프라인 장기 듀레이션 NPV) + 배당 2.93% 상대매력 → -100bp 시 멀티플 +1~2x, 주가 +6~9%. trontinemab·CT-388은 2030년대 현금흐름이라 금리 최대 민감.
  • IRA 약가협상 → 순가격·매출·유효특허수명 → Ocrevus(CHF4.9b, 2028~29 LOE 중첩)·Hemlibra 노출. 협상 지정 시 순가격 25~60% 인하, 메디케어 매출 두 자릿수 % 감소. 바이오 13년 룰로 소분자보다 보호는 김.
  • 미 관세(Section 232) → 마진(COGS) → 유럽·스위스산 수입에 15~25% 관세 시 그룹 마진 수백 bp 잠재 훼손. ~500억 달러 미국 투자 발표로 수년 걸쳐 상쇄·심리 헤지.
  • 비만 CT-388 → 밸류에이션(순수 상방 옵션) → 현재 기여 제로, 비대칭 상방·확률 낮음. 후발주자라 차별화 데이터가 관건, 하방 제한적.
  • HER2 등 LOE → 매출·마진 → HER2·Ocrevus·Hemlibra 합산 ~CHF18~19b(제약 매출 40%)가 2028~30 방어 국면. 시밀러 진입 후 1~3년 내 미국 매출 30~50% 급감, 디레이팅 근거.
  • 파이프라인 이벤트 → 밸류에이션 촉매 → evERA(2026.12) 단일 이벤트로 주가 두 자릿수 변동. 목표가 밴드 230~462의 넓이를 설명.
  • 중국 정책(VBP·반부패) → 진단 매출·마진 → VBP 편입 품목 가격 두 자릿수 % 인하. 제한적(진단 비중 ~22%만큼).
  • 치매 trontinemab → 밸류에이션(원거리 옵션) → 2030년대 현금흐름, 금리 하락과 결합 시 증폭. ARIA 이슈 시 옵션 소멸.
4. 가치평가 (레버 중심)

🟢 낙관 — 적정가치 CHF505 (밴드 450~580), 상방 +49% (+32~+71%)

  • 매출 CAGR 레버 — 10년 ~5.4%(61.5b→~105b, 7.0%→4.0% 감속). 빅3 침식 종료로 상단이 순수 신제품 성장으로 열림. CAGR 4.4→5.4%(+1%p) 시 적정가 +12~15%.
  • FCF마진 레버 — 19.6%→21.8%(코어영업 35%→~38%). 신제품 믹스 상향+고정비 레버리지. 20→21.8% 시 적정가 +8~10%. 이중계상 위험으로 +3%p로 절제.
  • WACC 레버(최강) — 6.5%(중립 7.0% 대비 -0.5%p). 저베타·스위스 저금리·39년 증배. 7.0→6.5% 시 448→506(+13%), 6.0%면 579.
  • 영구성장 g 레버 — 2.25%(중립 2.0% +0.25%p). 고령화·특허 재창출력. 2.0→2.5% 시 484→530(+9%).

5. 리스크
  • CHF 초강세 → 보고 매출·마진 이중 타격 + 비관 WACC 위험프리미엄 정당화. CER +7%가 보고 +2%로 희석.

6. 결론
  • 스탠스 — 적정~소폭 저평가(fairly valued with positive skew). "성장주"가 아니라 "무너지지 않음의 대가"를 지불받는 디펜시브로, 하방은 배당·35% 마진·CHF11.8b FCF가, 상방은 파이프라인이 담당하는 비대칭 보유.
  • 비중 — 저평가 매수가 아닌 '비대칭 옵션 보유'로 규정. 하방(Ocrevus/Hemlibra 급락+giredestrant 실패)은 상당분 프라이싱, 상방(비만·치매+마진 서프라이즈)은 미프라이싱.
  • 트리거① evERA FDA 결정(2026.12) → base(연착륙) 이상이면 저평가로 재평가, 19x·CHF370+ 경로.
  • 트리거② Ocrevus/Hemlibra 실측 침식속도 → 예상보다 빠르면 3%대 재평가, 관리되면 4~5% CAGR·멀티플 방어.
  • 트리거③ CT-388 비만 데이터 → 유효 데이터 시 현재가 미반영 옵션의 NPV 신규 반영, 가장 큰 PER 상방 레버.
  • 핵심 프레임 — 단기 변동성은 CHF·금리, 중기 방향은 LOE 공백 대 파이프라인·IRA, 장기 재평가 옵션은 비만·치매. 목표가 밴드 CHF230~462의 이례적 넓이가 방어주(15x·230)와 성장 재점화주(20x+·462) 사이 시장 인식을 반영.

  • 비만 TAM·컴파운딩·급여 → 매출 볼륨 채널 → 방향 +, 크기 최대(★★★★★). TAM 2030년대 초 $100~150b. 컴파운딩 종료로 회색시장→정품 전환 순풍.
  • 메디케어 비만 커버리지 → 볼륨↑·순가격↓ 양날. 급여 개방 시 수백만~천만 명 추가, 매출 순플러스·마진 희석.
  • MFN(최혜국 약가) → 순가격→매출·마진 동시 → 강한 마이너스, 잠재 크기 최대이나 저확률(★★★★). GLP-1($36.5b) 순가격 직접 타격 위험. LillyDirect 직판(월 ~$349~499)로 명분 약화·자가 완충.
  • IRA 협상 → 순가격→매출 → 마이너스이나 구조적 면역(★★). Jardiance·Trulicity·Verzenio는 어차피 특허절벽 임박. 하방 수 %p 이내.
  • 경쟁(노보 등) → 점유+순가격 동시 → 마이너스, 누증(★★★). TAM 성장이 경쟁 희석 흡수하는 한 감내. TAM 둔화 순간 가격전쟁 전환 임계.
  • 관세 232조 → COGS→마진/FCF → 중립~약방어(★★). $27b+ 미국 신규 제조로 리쇼어링 상쇄. 채널이 마진→FCF로 이동(capex $10b가 OCF 잠식).
  • 실물경기·고용 → 볼륨(현금환자) 채널 → 경기동행(★★). GLP-1 현금직판이 반쯤 소비재 성격, 신규 처방·순응률 경기 민감.
  • USD 환율 → 매출 환산 채널 → USD 강세 시 마이너스이나 둔감(★). 미국 매출 비중 높아 노출 작음.
4. 가치평가 (레버 중심)

🟢 낙관 (적정 $1,816, +52.7%)

  • 레버① 매출 CAGR 15.5% — FY26 +29%($84b)→점감→FY35 +6.5%, 매출 $84b→$275b. 시장 컨센(~12~13%) 상향분이 FCF·TV 기저를 동시 견인, 최대 상방 동인.
  • 레버② 영업마진 40%→47% — GM 80%+ 고정비에서 매출 3배 시 판관비·R&D 희석. 42% 평탄가정 대비 NOPAT +10%p.
  • 레버③ WACC 7.5% — 저베타·순부채 0.6x 정당화. TV가 EV의 77%라 0.5%p가 최대 지렛대(7.0%→$2,112, 8.0%→$1,587).
  • 레버④ 영구성장 3.5% — 만성질환 구조수요. 3.0%→$1,651, 4.0%→$2,029.
  • 레버⑤ capex 정상화 15%→5.8%·세율 15% — 후반 정상화가 FY30+ FCF 확대(FY35 FCF마진 39%), TV 기저 확대.
  • 주의 — TV가 EV의 77%. 노보 경쟁·가격압박·orfo 상업화가 경로 훼손 시 하방 탄력 큼.

🔴 비관 (BASE $327, -73%, 밴드 $150~560)

  • 레버① 성장 반토막(-$580, 최대) — 2036 매출 $257b→$126b. DCF는 성장에 볼록, 노보 추월·경구 저장벽·중단률로 성장 곡선 눕힘.
  • 레버② WACC 8%→9.5%(-$161, 2위) — TV 비중 47%로 할인율 극민감. 규제 프리미엄·단일 카테고리 56% 집중·배수 리레이팅.
  • 레버③ 영구성장 3%→1.5%(-$58) — 2036 절벽이 터미널 시점과 겹쳐 "영속 3%" 성립 불가.
  • 레버④ 마진 34%→30% 캡(-$92) — 넷 프라이싱 침식·믹스 악화(고마진 노후자산 만료)·경직적 투자로 영업레버리지 역전.
  • 레버⑤ 약가충격+조기 절벽(테일 -$120~170) — 매출 역성장·마진 17~23% 붕괴 시 $150~210(-83~87%).

🔵 시장평가 (역DCF: Priced-to-Perfection)

  • 레버① 매출성장(지배적) — 적정~보수적 — 암묵 CAGR ~13%가 FY26E 가이드 +26%보다 낮음. 부담은 5~10년차 지속성. 11→15% 상향 시 정당 EV +25%.

5. 리스크
  • 밸류 듀레이션 리스크 → EV의 47~77%가 먼 미래 TV. 금리·프리미엄 상승 시 성장주처럼 붕괴, 시장은 방어적 제약주로 오인.

  • 넷 프라이싱 vs 표시가 괴리 → 리베이트 경쟁으로 실현 넷마진 이미 낮고 추가 하락 여지, GM 80% 영속 가정 위협.
  • 터미널-특허절벽 시간 중첩 → 2036 물질특허 만료가 DCF 터미널 시점과 겹쳐 "영구 3% 성장" 구조적 불가.
  • MFN 정책 꼬리위험 → 저확률·고손실 비대칭. 실현 시 GLP-1 순가격 직접 타격, 매출 역성장까지 개방.
  • 높은 중단률·침투 과대추정 → 비만약 1년 내 상당수 이탈, 보험 커버리지 한계로 TAM 침투 곡선 과대평가.
  • 경쟁 재편 → 노보 ex-US 1위 추월, 암젠 MariTide·차세대 경쟁이 2027+ 듀오폴리 렌트 위협.
  • 비대칭 하방 → 완벽한 실행이 이미 가격에 반영. 노보식 미스 한 번에 31x→22~24x 디레이팅이 실적 미스와 곱해져 낙폭 증폭($850).
